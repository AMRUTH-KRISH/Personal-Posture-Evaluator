# 🏋️ AI Fitness Assistant – Personal Posture Evaluator

An AI-powered fitness web application that analyzes user posture in real time using computer vision and provides feedback, rep counting, and personalized fitness recommendations.

---

## 🚀 Features

- 🎯 Real-time posture detection using computer vision
- 📊 Accuracy-based scoring system
- 🔁 Rep counting for exercises
- 🧘 Yoga mode with hold timer and feedback
- 🔊 Voice guidance for better interaction
- 📈 Progress tracking with graphs
- 🥗 BMI-based diet & workout planner
- 👨‍💻 Admin dashboard for user management

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, Bootstrap
- Backend: Flask (Python)
- Computer Vision: OpenCV
- Database: SQLite
- Others: JavaScript, Chart.js
